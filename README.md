第一组 失控主题 2d跑酷游戏

1.玩法介绍
  主角获得了某种神秘能力，可以操纵时间，玩家带入主角视角，利用主角的超能力逃离这个神秘的房间。
  
2.游戏教程
  主角的所有操作只能在子弹时间内运行，按住shift进入子弹时间，然而主角的能力是不可控的，他必须在限定的时间内完成操作，逃离这里。
  
  shift + 1 控制角色在子弹时间内位移

  shift + 2 生成一个长方形实体，在此期间按下a/d进行旋转
  
  shift + 3 选择长方形实体并将其变形为弹性实体
  
  shift + 4 删除一个实体
  
  shift + 5 显示/隐藏一个仅存在于high tower的镜像实体。
